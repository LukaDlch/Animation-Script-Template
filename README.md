# Principle Animation Brief

A client-facing form for motion graphics projects. Helps clients describe their animation needs second by second, with visual format guides and automatic pricing.

## Features

- Project selection (Dexari/Alta) with dynamic theming
- Animation format templates (Ad, Announcement, Feature, Tutorial)
- Easing style previews (Linear, Easy Easing, Bounce)
- Segment-based timeline with dual-handle sliders
- Reference image upload and links
- Automatic price estimation
- PDF export

## Deployment

This is a static HTML file. Simply deploy to Vercel, Netlify, or any static host.

## Usage

1. Client fills out the form
2. Adds segments for each part of the animation
3. Clicks "Generate PDF Brief"
4. Sends PDF to animator
